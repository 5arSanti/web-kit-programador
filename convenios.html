Convenios

AWS Academy

AWS Academy ofrece recursos y cursos diseñados para estudiantes y educadores interesados en aprender sobre Amazon Web Services (AWS).

Proporciona acceso a contenido actualizado, laboratorios prácticos y certificaciones que preparan a los usuarios para carreras en computación en la nube.

Google Cloud Platform

Google Cloud Platform ofrece herramientas potentes para el almacenamiento, análisis de datos, aprendizaje automático y desarrollo de aplicaciones.

A través de su programa educativo, estudiantes y educadores pueden acceder a recursos formativos y créditos gratuitos para explorar y aprender sobre la plataforma.

Cisco Skills for All

Cisco Skills for All es una plataforma educativa gratuita de Cisco que brinda capacitación en habilidades tecnológicas esenciales, como redes, ciberseguridad y programación.

Está diseñada para preparar a los estudiantes para certificaciones reconocidas en la industria, con contenido accesible para principiantes.

GitHub Education

GitHub Education proporciona herramientas y recursos educativos para estudiantes y profesores interesados en el desarrollo de software.

A través de esta plataforma, los usuarios pueden acceder a repositorios de código, gestionar proyectos de software y colaborar en tiempo real, además de obtener acceso a software gratuito y descuentos en servicios relacionados con la programación.

Azure Portal Education

Azure Portal Education es una plataforma educativa de Microsoft Azure que permite a estudiantes y educadores explorar y aprender sobre la computación en la nube.

Proporciona acceso gratuito a recursos formativos, herramientas de desarrollo y créditos para utilizar los servicios de Azure, facilitando la adquisición de habilidades en tecnologías de nube.
